# -
Проект создание веб приложения для микроконтроллера ESP32, который создает точку доступа и после подключения набрав в браузере адрес 192.168.4.1 открывается локальный сайт для управления светодиодной лентой WS2812B. На сайте будут кнопки включения, выключения, ускорение скорости эффектов, задание общего цвета.
